Docker - uses docker compose:
---

docker compose up

You can check if its running by typing
docker ps

cAdvisor is also installed and is running on port 8080
https://github.com/google/cadvisor

Start
---

```
npm start
```

Setup
---

```
npm install
```

Compile
---

```
npm run compile
```
